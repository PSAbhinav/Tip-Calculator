# Tip Calculator App

## 📱 Overview
Tip Calculator is an Android application that calculates the tip amount based on the bill and selected percentage.

## ✨ Features
- Tip calculation
- Real-time updates
- Clean and simple UI

## 🛠️ Technologies Used
- Kotlin
- Jetpack Compose
- Android Studio

## 🚀 How to Run
1. Clone the repository
2. Open in Android Studio
3. Run the app

## 📚 Learning Outcome
This project demonstrates **user input handling and state management in Compose.**
